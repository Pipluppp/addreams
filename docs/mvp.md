# mvp

Functionality of addreams. Our value is in producing marketing/ads for products.
Generating ad/product images and shots, product marketing graphics, and product videos. 

No auth, profiles yet. Just the addreams brand landing page, features, workflows.

Attach API to qwen image/video models.

**Example workflows**

Given a reference image of the product, generate an image of the product framed
following the text prompts (ie., be it for photoshoots, graphics, social media)

Given a reference image of the product, generate a video of the product framed/directed
following the text prompts.

Generate a product image without reference directly from text prompts.
